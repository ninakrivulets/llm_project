# Запуск

Создайте `.env` файл (на примере .env.example), и укажите `OPENAI_API_KEY` и `OPENAI_PROXY`

Запуск осуществляется:
```bash
docker compose up --build -d
```

Остановка:
```bash
docker compose down
```
